- aes_128 module: This module implements the AES-128 encryption algorithm. It takes in a clock signal, state, key, and outputs the encrypted output. The module consists of several sub-modules that perform key expansion and encryption rounds. From a security perspective, the AES algorithm is considered secure and widely used in various applications.
- expand_key_128 module: This module is responsible for expanding the input key into round keys used in the AES encryption algorithm. It takes in a clock signal, input key, and a round constant. The module performs key expansion using XOR and shift operations. From a security perspective, the key expansion process is a critical part of the AES algorithm and should be implemented correctly to ensure the security of the encryption.
- S4 module: This module performs a substitution operation on a 32-bit input. It takes in a clock signal and input, and outputs the substituted value. From a security perspective, the substitution operation is a non-linear operation that adds confusion to the encryption process and enhances the security of the AES algorithm.
