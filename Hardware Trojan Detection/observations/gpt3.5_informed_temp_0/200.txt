The module takes in a clock signal, state, and key as inputs, and produces the encrypted output as the output. The module performs the encryption in multiple rounds, with each round using a different round key derived from the original key.
- module expand_key_128: This module expands the original key into multiple round keys using the AES key schedule. It takes in the clock signal, the original key, and a round constant as inputs, and produces two sets of round keys as outputs. The module uses a shift register and XOR operations to generate the round keys.
