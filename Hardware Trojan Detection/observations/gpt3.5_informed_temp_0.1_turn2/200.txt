- module expand_key_128: This module expands the input key into round keys using the AES key expansion algorithm. It takes in a clock signal, input key, and rcon as inputs, and produces two sets of round keys as outputs. The module uses a shift register and XOR operations to generate the round keys.

It takes in a reset signal, clock signal, and data as inputs, and produces a 20-bit counter as the output. The counter is updated on each clock cycle based on the input data and the LFSR feedback polynomial.
